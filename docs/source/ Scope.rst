SCOPE
----
----

UniSoc is a mobile application designed to improve student engagement with university societies at University of Portsmouth. The app centralises society discovery, event management, and member communication. It aims to solve the following problems:

Fragmented information across social media platforms.
Difficulty in tracking society events and attendance.
Inefficient communication between society admins and members.
Lack of a single interface for joining/leaving societies and managing notifications.

The system targets all students, including freshers, returning undergraduates, postgraduates, and society committee members (e.g., Presidents, Secretaries, Vice-Presidents)
